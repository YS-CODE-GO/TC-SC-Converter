# 中文简繁转换工具 (TC-SC Converter)

基于 Rust 开发的专业级中文简繁转换工具，旨在提供高效、安全且易用的批量转换体验。支持多线程加速、多种文件格式自动识别以及完善的 UI 交互。

## 功能特性

- **核心转换**：集成 `zhconv` 核心，支持繁体转简体（默认）及简体转繁体。
- **批量处理**：支持多文件/文件夹并行转换，充分利用多核性能。
- **智能识别**：自动检测文件编码（UTF-8, GBK 等），完美保留原文件结构（缩进、换行等）。
- **灵活模式**：
  - **替换模式**：直接修改原始文件。
  - **新建模式**：保留原文件，生成带时间戳的新文件（如 `file_20231227_123456.txt`）。
- **配置持久化**：自动记忆窗口大小、位置以及用户的转换选项（如转换模式、保存模式等）。
- **专业 UI 交互**：
  - **拖放增强**：支持文件/文件夹拖放，提供实时拖拽视觉反馈（全屏蒙版提示）。
  - **拖入即转**：可选开启“拖入即转换”功能，实现零点击操作。
  - **路径优化**：队列中路径采用缩略显示，鼠标悬停可查看完整路径，点击可快速打开所在文件夹。
  - **一键清理**：支持一键清空任务队列。
  - **实时预览**：内置多行文本输入框，支持实时简繁转换预览。
- **全格式支持**：支持包括文本、代码、配置文件等在内的 20+ 种文件后缀。
- **高性能与安全**：基于 Rust 编写，无 C++ 依赖，内存安全且运行高效。

## 支持的文件格式

支持但不限于以下后缀：
- **文档**：`.txt`, `.md`, `.html`, `.htm`
- **代码**：`.js`, `.ts`, `.py`, `.c`, `.cpp`, `.h`, `.hpp`, `.java`, `.cs`, `.rs`, `.go`
- **配置/脚本**：`.json`, `.xml`, `.yaml`, `.yml`, `.ini`, `.toml`, `.sh`, `.bat`, `.ps1`

## 使用说明

1. **文本即时转换**：
   - 在上方文本框输入或粘贴文字。
   - 开启“自动转换预览”或点击“立即转换文本”即可完成转换。
2. **批量文件处理**：
   - 将文件或文件夹拖入窗口，或通过 `文件 -> 添加文件` 选择。
   - 配置转换方向（繁/简）与保存模式（替换/新建）。
   - 可选开启“拖入即转换”以简化流程。
   - 点击“开始批量转换”启动任务。
3. **任务管理**：
   - 在队列中查看实时进度、耗时及状态。
   - 使用“🗑 清空队列”按钮快速重置。
   - 鼠标悬停在路径上查看完整位置，点击路径打开资源管理器。

## 技术规范

- **开发语言**：Rust 1.75+
- **GUI 框架**：egui / eframe (0.29)
- **转换引擎**：zhconv (支持 MediaWiki & OpenCC 规则)
- **并发处理**：Rayon + Std Thread
- **编码检测**：chardetng + encoding_rs
- **配置存储**：基于 eframe 的本地持久化存储
